## Quick Start

Get up and running with TUI-Kit-AI in minutes:

<CodeGroup>

```bash npm
npm install @tui-kit-ai/core @tui-kit-ai/ai @tui-kit-ai/agents
```

```bash yarn
yarn add @tui-kit-ai/core @tui-kit-ai/ai @tui-kit-ai/agents
```

</CodeGroup>

### Basic Chat Application

```typescript
import { useTerminal, Box, TextInput } from '@tui-kit-ai/core';
import { ChatContainer, AIService } from '@tui-kit-ai/ai';

const { screen } = useTerminal();

const aiService = new AIService({
  provider: 'openai',
  model: 'gpt-4',
  apiKey: process.env.OPENAI_API_KEY
});

const chatContainer = new ChatContainer({
  parent: screen,
  messages: [],
  onMessageSubmit: async (content) => {
    const result = await aiService.streamCompletion([
      { role: 'user', content }
    ]);
    
    for await (const chunk of result.textStream) {
      // Handle streaming response
    }
  }
});

screen.render();
```
