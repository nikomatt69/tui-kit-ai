PDF processing, text extraction, search, and analysis

üìã Component Information

Operations: Load, View pages, Extract text/tables/images, Search content

Features: Document list, Page viewer, Search results, Metadata display

Formats: PDF documents with text, images, tables, and forms

üéÆ Controls

Document: Select a sample PDF document to load

Load Document: Load the selected document into the viewer

Extract Text: Extract and display all text content

Extract Tables: Extract and display table data

Extract Images: Extract and display image information

Search Content: Search for specific text in the document

Clear Document: Clear the current document and reset

üíª Usage Example

import { PDFTool } from '@tui-kit-ai/ai';

const pdfTool = new PDFTool({
  showDocumentList: true,
  showPageViewer: true,
  showSearchResults: true,
  onDocumentLoad: (doc) => console.log('Document loaded:', doc),
  onTextExtract: (text) => console.log('Text extracted:', text),
  onSearch: (results) => console.log('Search results:', results)
});

// Load PDF document
pdfTool.loadDocument('path/to/document.pdf');

// Extract content
const text = pdfTool.extractText();
const tables = pdfTool.extractTables();
const images = pdfTool.extractImages();

// Search content
const results = pdfTool.search('search term');

// Get document metadata
const metadata = pdfTool.getMetadata();
