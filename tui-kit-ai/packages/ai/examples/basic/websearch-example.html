üìã Component Information

Engines: Google, Bing, DuckDuckGo, Yahoo, Brave, Custom

Features: Search bar, Filters, History, Results with metadata

üéÆ Controls

Toggle Filters: Show/hide advanced search filters

Toggle History: Show/hide search history

Clear History: Clear all search history

Simulate Search: Perform a mock search with sample results

üíª Usage Example

import { WebSearchTool } from '@tui-kit-ai/ai';

const webSearch = new WebSearchTool({
  defaultEngine: 'google',
  showFilters: true,
  showHistory: true,
  maxResults: 10,
  onSearch: (query, results) => console.log('Search results:', results),
  onResultClick: (result) => console.log('Result clicked:', result)
});

// Perform search
webSearch.search('artificial intelligence', {
  engine: 'google',
  type: 'webpage',
  filters: {
    language: 'en',
    region: 'us',
    dateRange: 'past_year'
  }
});

// Get search history
const history = webSearch.getHistory();
console.log('Search history:', history);
