🎨 ImageAI Component Example

AI image generation and analysis with multiple models

📋 Component Information

Generation Models: DALL-E 2/3, Midjourney, Stable Diffusion, Imagen, Parti

Analysis Models: GPT-4 Vision, Claude 3 Vision, Gemini Vision, LLaVA

Features: Prompt editor, Image gallery, Generation history, Object analysis

🎮 Controls

Mode: Switch between image generation and analysis

Model: Select AI model for generation or analysis

Generate Image: Create a new image from text prompt

Analyze Image: Analyze an uploaded or generated image

Clear Gallery: Clear all images from the gallery

Toggle History: Show/hide generation and analysis history

💻 Usage Example

import { ImageAI } from '@tui-kit-ai/ai';

const imageAI = new ImageAI({
  defaultModel: 'dall-e-3',
  showPromptEditor: true,
  showGallery: true,
  showHistory: true,
  onImageGenerate: (image) => console.log('Image generated:', image),
  onImageAnalyze: (analysis) => console.log('Image analyzed:', analysis)
});

// Generate image
imageAI.generateImage('A futuristic cityscape at sunset', {
  model: 'dall-e-3',
  size: '1024x1024',
  quality: 'hd'
});

// Analyze image
imageAI.analyzeImage(imageUrl, {
  model: 'gpt-4-vision',
  detail: 'high'
});

// Get generation history
const history = imageAI.getHistory();
