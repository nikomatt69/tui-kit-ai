üìö About These Examples

These examples demonstrate the core functionality of each AI component in the @tui-kit-ai/ai package. Each example includes:

  • Interactive Controls: Buttons and inputs to test different features
  • Real-time Updates: Live demonstration of component behavior
  • Mock Data: Realistic sample data for testing
  • Usage Examples: Code snippets showing how to use each component

üéØ Key Features Demonstrated

  • Stream: Real-time data streaming, progress tracking, chunk management
  • CodeBlock: Syntax highlighting, line numbers, multiple themes
  • TokenInfo: Token counting, cost analysis, model information
  • ModelSelector: Model comparison, filtering, provider selection
  • WebSearchTool: Multi-engine search, filters, result metadata
  • PDFTool: Document loading, text extraction, content search
  • ImageAI: Image generation, analysis, prompt editing

üöÄ Getting Started

To use these components in your own project:

  1. Install the package: npm install @tui-kit-ai/ai
  2. Import the components: import { Stream, CodeBlock, TokenInfo } from '@tui-kit-ai/ai'
  3. Configure and use: const stream = new Stream({ variant: 'realtime' })
