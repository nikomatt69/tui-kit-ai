Real-time streaming with progress tracking and statistics

üìã Component Information

Variant: realtime

Features: Progress tracking, Statistics, Chunk management

States: idle, connecting, streaming, paused, completed, error

üéÆ Controls

Start: Begin streaming with mock data

Pause/Resume: Control streaming state

Stop: End streaming and show final stats

Clear: Reset all data and statistics

üíª Usage Example

import { Stream } from '@tui-kit-ai/ai';

const stream = new Stream({
  variant: 'realtime',
  showProgress: true,
  showStats: true,
  onChunk: (chunk) => console.log('New chunk:', chunk),
  onStateChange: (state) => console.log('State:', state)
});

// Add data chunks
stream.addChunk({
  type: 'text',
  content: 'Hello, World!',
  metadata: { source: 'api' }
});

// Control streaming
stream.start();
stream.pause();
stream.resume();
stream.stop();
